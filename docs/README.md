# openCenter Documentation

Welcome to the official documentation for `openCenter`. This site provides a comprehensive set of resources to help you understand, use, and contribute to the project.

The documentation is organized into four main sections, following the [Diátaxis framework](https://diataxis.fr/):

*   **Tutorials**: Learning-oriented lessons to get you started.
*   **How-To Guides**: Goal-oriented recipes for common tasks.
*   **Reference**: Factual, technical descriptions of the machinery.
*   **Explanation**: Conceptual, high-level discussions of the project's architecture and design.

---
